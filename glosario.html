TIPOS DE ETIQUETAS

En HTML y CSS tenemos diferente tipos de etiquetas cada uno con un propósito en particular una etiqueta se puede componerse de dos maneras.

ESTRUCTURA DE ETIQUETAS
Etiquetas que solo se apertura Etiquetas que tienen una apertura y un sierre
Este tipo de etiqueta común mente se utilizan para indicar una dirección u origen se componen por argumento u opciones tales como src href. Este tipo de etiquetas común mente se utilizan para encerrar un contenido tales comotexto img entre otros.

➕La cabeza de cada pagina web tiene que contener:

<html> </html> Es lo que encierra todo el codigo de la pagina.
<title> </title> El nombre que llevará la pagina.
<body> </body> Su principal funcion es hacer un cuerpo para la pagina.
<header> </header> El titulo que llevará la pagina.

➕Las diferentes maneras de hacer un lista de son las variables de:

<ul> </ul> Informar al equipo que se hará un listado.
<li> </li> Un tipo de listador con un punto completar la información.

➕Para poder llegar a hacer una tabla con sus propiedades se necesita:

<table> </table> La tabla en sí.
<tr> </tr> Las columnas que se quiere agregar.
<td> </td> Las casillas que se desean agregar y su contenido.

➕Al crear un formulario despenden de la funcionalidad de las siguientes variables.

<label> El texto que se muestra en la barra
<imput> Lo que se quiere agregar para continuar, en este caso es un texto.
<form> Encierra todo lo antes mecionado.

➕Otras etiquetas que a pesar que no tengan tanto protagonismo son bastante utiles para editar la pagina web
